# 🤖 Automação de Cadastro de Produtos
Projeto desenvolvido em Python para automatizar o processo de cadastro de produtos em um sistema web da empresa, utilizando automação de teclado e mouse.

# 📌 Sobre o Projeto
Este projeto tem como objetivo automatizar o cadastro de produtos em um sistema online, evitando o trabalho manual repetitivo.
A automação realiza:
Abertura do navegador
Login no sistema
Leitura da base de dados (produtos.csv)
Cadastro automático de cada produto
Repetição do processo até finalizar a lista

# 🛠️ Tecnologias Utilizadas
🐍 Python 3
🤖 PyAutoGUI
📊 Pandas
📄 CSV (base de dados)
🌐 Navegador Google Chrome

# 🧠 Conceitos Aplicados
Automação de tarefas repetitivas
Manipulação de dados com Pandas
Leitura de arquivos CSV
Controle de tempo com time.sleep()
Estruturas de repetição (for)
Integração entre software e interface web

# 🎯 Objetivo do Projeto
Este projeto foi desenvolvido para:
Praticar automação com Python
Trabalhar com manipulação de base de dados
Simular um cenário real de empresa
Aplicar lógica de programação em um problema prático
